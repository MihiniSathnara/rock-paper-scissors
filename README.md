# Rock Paper Scissors Game

A web-based Rock Paper Scissors game where a player competes against a bot. The game keeps track of scores and declares a winner when either the player or the bot reaches 5 points.

- Player selects Rock, Paper, or Scissors.
- Bot randomly selects a move.
- The choice of both player and bot is displayed using images.
- Scores are updated in real-time.
- First to reach 5 points wins the game.
